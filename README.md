# Chess Game

A modern 3D chess experience built with Unity, featuring realistic graphics, glassmorphism UI, and AI opponent.

## Features
- **3D Chess Board** with detailed Staunton pieces rendered in glass-like style.
- **Dynamic Environment**: Play on a park‑like backdrop with realistic lighting.
- **Glassmorphism UI** overlay for a sleek, premium look.
- **AI Opponent**: Play against a computer‑controlled black side.
- **Sound Effects** for moves, captures, and checkmate.
- **Smooth Animations** for piece movements.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Chess-game.git
   ```
2. Open the project in Unity (2022.3 or later).
3. Resolve any package dependencies via the Unity Package Manager.
4. Press **Play** in the Unity editor to start the game.

## Running in Browser

This project also includes a web version built with JavaScript. To run it locally in your browser:

1. Ensure you have **Node.js** (v18 or later) and **npm** installed.
2. Open a terminal in the project root directory:
   ```bash
   cd path/to/Chess-game/web
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173` (or the URL shown in the terminal) to view the game.

The web version mirrors the core gameplay and UI, allowing you to play directly in the browser without Unity.


## Usage
- Use the mouse to select and move pieces.
- The AI will automatically make moves for the black side.
- Enjoy the immersive visuals and sound effects.
